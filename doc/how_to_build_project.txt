#summary How to build the project from source code.

  * Install software
  * Checkout the source code
  * Build the distributive in command-line mode

= Install software =

 1) Eclipse 3.6.2 
 2) Maven 3: http://maven.apache.org/download.html

= Checkout the source code =
get Redmine Java API from here https://redmine-java-api.googlecode.com/svn/trunk

= Build the distributive in command-line mode =

make sure you have Maven's "bin" folder in your path.
Run this command in the project root folder:
  mvn clean package
This will create distributives in "_project_root\build_" folder.
