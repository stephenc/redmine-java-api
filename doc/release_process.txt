Release process for Redmine Java API.
-------------------------------------

Change version number in   \org.redmine.ta.api\pom.xml

Commit to SVN, wait for the new auto-build. Verify all tests pass.

Upload the build artifact ("build\redmine-api-xxxx.zip") to Google Code webpage:
https://code.google.com/p/redmine-java-api/downloads/list  

Copy the p2 repository artifacts from "org.redmine.ta.site\target\repository" to ".\repository" folder. 
Commit ".\repository" folder to SVN, this repository will be accessible to everyone 
through http://redmine-java-api.googlecode.com/svn/trunk/repository URL
