Redmine Java API.

USAGE.

1. Install the license using "licenses.cmd" (for Windows) or "licenses.sh" script (Linux)
    Sample command:
    licenses.cmd <path_to_license_file>
   
    Redmine Java API will work in "trial" mode if no license is installed. 
    See http://www.taskadapter.com website for information about the trial mode. 
 
     
2. Add the Redmine API Jar and all jars from the subfolders in "lib" folder to the classpath
  of your Java application.

3. See sample code in the "samples" folder.
